# UE-Free ISAC Pedestrian Tracking using Sionna RT

A UE-free integrated sensing and communication (ISAC) framework for pedestrian localization using prior channel information generated from a digital twin in Sionna RT.

---

## Framework

<p align="center">
  <img src="images/fig3_framework.png" width="850"/>
</p>

The proposed framework first constructs a prior channel database from a digital twin in the offline phase. During the online phase, the prior channel is subtracted from the measured channel to extract pedestrian-induced paths, followed by TOA-based multilateration for pedestrian localization.

## Simulation Parameters

| Parameter | Value |
|-----------|-------|
| Carrier Frequency | 3.5 GHz |
| Base Stations | 7 quasi-monostatic BSs |
| Antenna | 1×1 isotropic |
| Target | Pedestrian |
| Prior Channel | LoS + Specular |
| Measured Channel | LoS + Specular + Diffuse |
| Ray Tracing Depth | 3 |
